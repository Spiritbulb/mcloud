# Menengai Cloud

A platform for Kenyan merchants and developers to build, automate, and grow online.

## What it does

- Website builder with pre-built React components
- Storefront hosting for merchants
- Organization-based workspace management
- Workflow automation via hosted n8n (coming soon)

## Stack

- Next.js
- PostgreSQL
- n8n (self-hosted, containerized)

## Status

Beta — active development.
